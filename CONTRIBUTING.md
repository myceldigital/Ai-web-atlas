# Contributing

AI Web Atlas is evidence-first. Contributions should improve the map, data, or methodology without turning the repository into a promotional landscape slide.

## High-value contributions

- Correct a company classification with public evidence.
- Replace an uncertain node with a sourced node.
- Improve map layout while preserving the visual grammar.
- Add structured fields to `data/companies.json`.
- Propose a new atlas version with a clear scope and evidence standard.

## Evidence standard

Preferred sources include company announcements, product documentation, public customer case studies, procurement notices, filings, earnings-call transcripts, and credible reporting.

Weak evidence includes unsourced social posts, anonymous comments, visual proximity on another map, and logo presence without explanatory text.

## Pull request checklist

- Explain the change.
- Link public evidence.
- Update `data/companies.json` if nodes changed.
- Update Mermaid source and rendered SVG if the map changed.
- Preserve ambiguity where the evidence is ambiguous.

## What will usually be closed

- Unsourced logo additions.
- Promotional company copy.
- Speculative customer claims.
- Attempts to remove valid criticism because it is unflattering.
- Quotes attributed to real people who did not say them.
