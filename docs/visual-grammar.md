# Visual Grammar

The atlas uses visual conventions to separate evidence, critique, and infrastructure.

| Convention | Meaning |
|---|---|
| Light box | Original poster category or confirmed ecosystem node |
| Orange dashed box | Uncertain reconstruction or educated guess |
| Blue box | Panel addition |
| Red box | Defense or national-security expansion |
| Dark substrate band | Physical dependency, not customer relationship |
| Strikethrough | Challenged or removed classification |
| Heavy border | Structural hub |
| Dotted link | Indirect or uncertain relationship |

Positioning is interpretive. It indicates category clustering, not verified market distance.
