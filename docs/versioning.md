# Versioning

Each atlas version should be preserved as its own map source, debate note, and findings record.

## Rules

- Do not silently overwrite old maps.
- Add a new version when the structure changes materially.
- Update `CHANGELOG.md` for every new version.
- Update `data/companies.json` when nodes are added, removed, or reclassified.
- Add evidence links whenever a confidence level is raised.

## Naming convention

- `maps/source/vN-short-name.mmd`
- `debates/vN-short-name.md`
- `findings/topic-name.md`
