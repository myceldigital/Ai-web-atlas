# Code of Conduct

Be rigorous, charitable, and evidence-led.

Contributors are expected to:

- critique claims, not people;
- preserve uncertainty when evidence is incomplete;
- avoid harassment, abuse, or personal attacks;
- disclose conflicts when proposing promotional additions;
- treat public correction as a strength of the project.

Maintainers may remove contributions or comments that are abusive, promotional, deceptive, or not aligned with the evidence standard.
