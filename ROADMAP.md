# Roadmap

This repository is designed to evolve as a versioned atlas rather than a one-off graphic.

## v4 — Browser agents, identity, and payments

Refine the application layer around browsers, wallets, payments, identity, permissions, and user-mediated agent actions.

## v5 — Sovereign AI and regional agent stacks

Separate the US, European, Chinese, Middle Eastern, and Asian agent-infrastructure ecosystems.

## v6 — Security, evals, and observability

Map the agent safety stack: sandboxing, browser isolation, tool permissions, evals, monitoring, audit trails, and incident response.

## v7 — Physical substrate deep dive

Expand power, datacenters, chips, networking, cooling, water, minerals, and export controls into a dedicated infrastructure atlas.

## Ongoing

- Replace uncertain nodes with sourced classifications.
- Keep `data/companies.json` synchronized with map files.
- Preserve version history instead of silently overwriting old interpretations.
