# Bibliography and Source Log

This file records the public source categories used by the atlas. Every material factual claim should eventually point to a specific source URL.

## Core source categories

- Parallel Web Systems announcements and product pages
- Official protocol documentation, especially Model Context Protocol documentation
- Public bot-traffic and web-infrastructure reports
- Public customer case studies and partner pages
- Defense technology company announcements
- Semiconductor and datacenter infrastructure sources
- Public filings, procurement records, and earnings-call transcripts where available

## Source quality hierarchy

1. Primary company or government document
2. Regulatory filing or public procurement record
3. Credible reporting with named sources
4. Product documentation
5. Public blog post or announcement
6. Unsourced secondary commentary

## Maintenance note

When adding or reclassifying a node, add the source URL to `data/companies.json` and explain the classification in the relevant finding or debate note.
